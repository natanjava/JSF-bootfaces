About the System

This system (version 2.0) registers users, launches and companies.

Each logged in user has different privileges. Each new user registered in the system can log in with username/password. Those who have privileges to register new launches need to say which partner company will be associated whenever a new launch is registered. The launch will be analyzed by an administrator. If the launch is rejected, the ADMINISTRATOR must explain why. And when there is new launch that have not been analyzed, a warning message will appear whenever an ADMINISTRATOR accesses the first page of the system.

ADMINISTRATOR has access to all parts of the system: User, Launch, Company, Review and Reports pages, in terms of insertion, update and deletion.
MANAGER can: Insert Launches. Once an ADMINISTRATOR approves/rejects a launch, MANAGER can no longer edit or delete it.
ASSISTENT can: Insert, edit and delete Partner Companies.
ALL USERS can: acess the list of users, launches and partner companies that have been saved in the system. Generate reports on users, launches and companies.
