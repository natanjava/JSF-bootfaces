About the System

This system (version 2.0) registers users, orders and companies.

Each logged in user has different privileges. Each new user registered in the system can log in with username/password. Those who have privileges to register new orders need to say which partner company will be associated whenever a new order is registered. The order will be analyzed by an MANAGER. If the order is rejected, the MANAGER must explain why. And when there is new order that have not been analyzed, a warning message will appear whenever an ADMINISTRATOR or MANAGER accesses the first page of the system (user page).

ADMINISTRATOR has access to all parts of the system: User, Order, Company, Review and Reports pages, in terms of insertion, update and deletion.
MANAGER can: analyze Order. Once an Order is approved/rejected by the MANAGER/ADMINISTRATOR, it is no longer possible to uptade, only delete.
ASSISTENT can: Insert, edit and delete Partner Companies.
ALL USERS can: acess the list of users, orders and partner companies that have been saved in the system. Generate reports on users, orders and companies.
